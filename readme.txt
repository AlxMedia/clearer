== Description ==

Clearer is a flexible blog and magazine theme with plenty of options. Optimized for all devices, this theme will stand out with a unique design, including dark/light mode. Demo: http://demo.alx.media/x/?theme=Clearer

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Clearer in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
